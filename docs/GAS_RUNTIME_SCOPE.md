# GAS Runtime Scope

Current scope is full Google Apps Script.

Audit dan perubahan harus fokus pada:

- `Code.js` sebagai backend GAS
- `Index.html`, `app.html`, dan `style.html`
- integrasi `google.script.run`
- dependensi Google Sheets
- Apps Script deployment behavior
- mobile camera / barcode runtime behavior
- browser permission dan iframe behavior di deployment `script.google.com`
